# Інструкція зі встановлення та збірки (Installation & Build Guide)

Цей документ містить покрокові інструкції щодо запуску та компіляції **Batch Image Editor** на операційних системах Windows 11, Linux (Manjaro/Ubuntu) та macOS.

---

## 🛠️ Локальний запуск із вихідного коду

### 1. Перевірка оточення
Переконайтеся, що на вашому комп'ютері встановлено **Python 3.10** або новішої версії. Перевірити версію можна командою:
```bash
python --version
```

### 2. Клонування або копіювання файлів
Переконайтеся, що у вашій робочій папці знаходяться такі файли:
* `main.py` — Точка входу.
* `editor.py` — Головне вікно та шорткати.
* `image_widgets.py` — Рамка кадрування та обробка миші.
* `image_service.py` — Математика пікселів (кроп, повороти, інверсія).
* `dark.css` — Стилі темної теми.
* `light.css` — Стилі світлої теми.

### 3. Встановлення залежностей
Відкрийте термінал / командний рядок у папці проєкту та виконайте:
```bash
pip install -r requirements.txt
```

### 4. Запуск додатка
```bash
python main.py
```

## 📦 Компіляція у самостійний виконуваний файл

Для збірки використовується утиліта `PyInstaller`, яка автоматично збирає інтерпретатор та залежності Qt в один пакунок.

> ⚠️ **Важливо:** PyInstaller не є крос-компілятором. Щоб отримати `.exe` для Windows — запускайте збірку на Windows. Щоб отримати додаток для Linux — запускайте на Linux.

### Команди для збірки:

* 🪟 **Windows (Збірка в `.exe`):**
  Відкрийте командний рядок (cmd) або PowerShell у папці проєкту та виконайте:
  ```bash
  pyinstaller --noconfirm --onefile --windowed --name="BatchImageEditor" --add-data "dark.css;." --add-data "light.css;." main.py
  ```

* 🐧 **Linux / Manjaro (Збірка в бінарний файл):**
  Відкрийте термінал та виконайте команду:
  ```bash
  pyinstaller --noconfirm --onefile --windowed --name="BatchImageEditor" --add-data "dark.css:." --add-data "light.css:." main.py
  ```

* 🍎 **macOS (Збірка в `.app` пакет):**
  ```bash
  pyinstaller --noconfirm --onefile --windowed --name="BatchImageEditor" --add-data "dark.css:." --add-data "light.css:." main.py
  ```

### 🔍 Особливості прапорців збірки:
* `--onefile` — пакує все в один файл.
* `--windowed` — приховує чорне вікно консолі при запуску.
* `--add-data` — **Критично важливо!** Вшиває файли стилів `dark.css` та `light.css` всередину виконуваного файлу, щоб теми коректно перемикалися на комп'ютерах користувачів. *(Зверніть увагу: у Windows використовується крапка з комою `;`, а в Linux/Mac — двокрапка `:`).*

### 📁 Результат збірки
Готовий до використання додаток з'явиться у створеній папці **`dist/`**. Тимчасові папки `build/` та файл `.spec` можна видалити.
