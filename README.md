# Batch Image Editor (PyQt6)

Сучасний, швидкий та адаптивний десктопний додаток на Python для **пакетної обробки та розумного кадрування зображень**. Інструмент розроблений для автоматизації рутинної роботи: ви налаштовуєте параметри на одному зображенні-шаблоні, а додаток миттєво застосовує зміни до всієї пачки файлів із збереженням їх оригінальних назв.

## ✨ Основні можливості (Features)
* **Розумний пакетний конвеєр**: Завантажуйте десятки зображень одночасно. Зручний інтерактивний список зліва дозволяє перемикатися між файлами та оцінювати прев'ю.
* **Монолітна рамка кадрування**: Кастомний Label з 8 маркерами руху. Рамка прив'язана до математичного центру картинки, завдяки чому вона не злітає та не стрибає при різкому ресайзі вікна.
* **Повноцінна історія дії (Undo/Redo)**: Кроки назад через `Ctrl + Z` та кроки вперед через `Ctrl + Shift + Z` працюють синхронно як для ефектів кольору, так і для мікро-рухів рамки мишкою.
* **Сучасний слайдер тем**: Інтерфейс оснащено сегментованим повзунком «Місяць/Сонечко» у верхній панелі.
* **Кросплатформний дизайн (QSS)**: Стилі повністю винесені в окремі файли ефектів, що дозволило обійти системні обмеження Linux KDE Plasma (Breeze) та Windows 11. На світлій темі всі контури чітко видно, а текст на кнопках автоматично стає контрастним.
* **Підтримка BMP без спотворень**: Примусова конвертація матриці пікселів у формат RGB32 захищає BMP файли від появи брудної жовтизни при інверсії.

## 📐 Геометрія інтерфейсу (Layout)
Додаток має жорсткий мінімальний розмір **1280x720 (HD стандарт 16:9)**, що запобігає деформації елементів:
* **Верхня панель**: Об'єднує всі системні дії (кнопку відкриття пачки, слайдер тем, ProgressBar збереження та кнопку «Зберегти»). Кнопки вирівняні строго по ширині панелей під ними.
* **Ліва панель**: Динамічний лічильник кількості файлів та інтерактивний список черги.
* **Центр**: Область графічного редактора та прев'ю.
* **Права панель**: Відокремлена міні-панель творчих ефектів (Інверсія, Поворот на 90°, Смивання налаштувань).

## ⌨️ Уніфіковані гарячі клавіші (Shortcuts)
Для максимальної зручності та виключення випадкових натискань усі гарячі клавіші стандартизовані під роботу з модифікатором **`Ctrl`**:

* `Ctrl + O` — Відкрити файли (викликати діалог вибору).
* `Ctrl + S` або `Ctrl + Enter` — Запустити пакетне збереження всієї пачки.
* `Ctrl + Z` — Скасувати останню дію (Undo).
* `Ctrl + Shift + Z` — Повернути скасовану дію (Redo).
* `Ctrl + I` — Увімкнути / вимкнути інверсію кольорів.
* `Ctrl + R` — Повернути поточний шаблон на 90 градусів.
* `Ctrl + Delete` або `Ctrl + Backspace` — Повністю скинути всі ефекти та рамку кадрування до оригіналу.
* `Стрілка Вгору` / `Стрілка Вниз` — Швидка навігація по списку файлів (зміна активного прев'ю).
* `Стрілка Вліво` / `Стрілка Вправо` — Швидке перемикання теми оформлення (Темна `🌙` / Світла `☀️`).
